# /kaiten-boards -- Управление канбан-досками

## Алгоритм

1. Вызови `get_boards` для получения списка досок
2. Вызови `get_columns` для получения колонок выбранной доски
3. Вызови `get_cards` для получения карточек с доски
4. Вызови `create_card` для создания новой карточки
5. Вызови `update_card` для обновления карточки
6. Вызови `get_tags` / `add_tag_to_card` для работы с тегами
7. Вызови `get_users` / `get_current_user` для пользователей
8. Вызови `get_spaces` / `get_space` для пространств

## Формат ответа

```
## Доски Kaiten

### Доска: Разработка
Колонки: Бэклог | В работе | Готово
- [42] Карточка 1 -- В работе -- Иванов
- ...
```

## Примеры

```
/kaiten-boards список досок
/kaiten-boards карточки на доске 123
/kaiten-boards создать "Новая задача" на доске 123 колонка 456
/kaiten-boards мои карточки
/kaiten-boards колонки доски 123
```
